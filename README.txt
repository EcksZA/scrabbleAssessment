--------------
Scrabble Score
--------------

Scrabble Score is a webpage that allows users to input a word into
the webpage and get an out put of the scrabble score for that 
particular word.

Author
------
Xolani Mfuko


Operating Instruction
---------------------
Input any word in the input box of the page, click on the "score"
button and the score for that word will be outputted.
